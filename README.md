# Love Percentage Calculator

A fun web application where users enter their name and their crush’s name to discover their **love compatibility percentage**.  
Built as a light-hearted project to practice frontend development and create an interactive user experience.

## 🚀 Features

- Enter your name and your crush’s name
- Instantly calculates a random/fun love compatibility percentage
- Simple and interactive user interface
- Responsive design for desktop and mobile devices
- Smooth user experience with instant results

## 🛠️ Tech Stack

- **HTML** – Structure of the website  
- **CSS** – Styling and responsive design  
- **JavaScript** – Logic for generating love percentage and interactivity  

## 📂 Project Structure

```bash
Love-Percentage-Calculator/
│
├── index.html        # Main webpage
├── style.css         # Styling file
├── script.js         # Love percentage logic
└── README.md         # Project documentation
```

## ⚙️ How It Works

1. User enters their name  
2. User enters their crush’s name  
3. The application processes the input  
4. A compatibility percentage is generated  
5. The result is displayed instantly on the screen  

Example:

```text
Your Name: Alex
Crush Name: Emma

❤️ Love Percentage: 87%
```

## 💡 Purpose of the Project

This project was built to:

- Practice frontend web development  
- Learn DOM manipulation using JavaScript  
- Build an interactive mini-project for portfolio/GitHub  
- Experiment with user input handling and dynamic output generation
